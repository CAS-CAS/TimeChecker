# TimeChecker

## 概要

* 時間範囲指定（例えば11時から22時）したあとに、ある時間（例えば14時）を指定して、ある時間が時間範囲に入っているかどうかを確かめる。
* 入力できるのは0時から23時まで。
* 日をまたいだ範囲（例えば21時から5時）でも動作する。
* 前述の例だと、21時は範囲内で、5時は範囲外。
* ただし、21時から21時のように設定したら21時だけ範囲内。

## 使い方

### 時間範囲の指定方法
* 「(開始時間) (終了時間)」のように入力
* 例: 6 17

### 時間の指定方法
* 調べたい時間を入力
* 例: 7

### 終わり方
Ctr + D
